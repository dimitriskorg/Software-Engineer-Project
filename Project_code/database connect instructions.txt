Στον παρακάτω σύνδεσμο κατεβάζετε το Connector/J ως platform independent

https://dev.mysql.com/downloads/connector/j/

Αφότου το κατεβάσετε, παίρνετε το αρχείο .jar που έχει και θα πρέπει να το βάλετε στον 
φάκελο lib του project που κάνατε και γράφετε κώδικα.

Τέλος, στο VS code δεν πατάτε απλά το Run Code αλλά στο βελάκι και μετά Run Java.

!!! Να αλλάξετε τα στοιχεία της βάσης ώστε να συμβαδίζει με τα δικά σας !!!
